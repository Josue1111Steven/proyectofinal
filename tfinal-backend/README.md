# Backend TFINAL

Backend mínimo en Node.js + Express para la Tarea Final.

## Requisitos
- Node.js 18+ y npm

## Instalación
```bash
npm install
```

## Ejecución
```bash
node server.js
```
Esto arrancará el servidor en:
```
http://localhost:3000
```

## Endpoints
- `POST /auth/login` → { token }
- `GET /productos`
- `POST /productos`
- `GET /productos/:id`
- `PUT /productos/:id`
- `DELETE /productos/:id`
